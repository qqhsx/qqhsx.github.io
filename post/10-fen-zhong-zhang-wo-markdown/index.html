    10分钟掌握Markdown

    前言

    写过博客或者github上面的文档的，应该知道Markdown语法的重要性，不知道的朋友们也别着急，一篇博客轻松搞定Markdown语法。话说这个语法超级简单，一看就会，不信你点进来看看。

    快捷键

    功能 快捷键
    加粗 Ctrl + B
    斜体 Ctrl + I
    引用 Ctrl + Q
    插入链接 Ctrl + L
    插入代码 Ctrl + K
    插入图片 Ctrl + G
    提升标题 Ctrl + H
    有序列表 Ctrl + O
    无序列表 Ctrl + U
    横线 Ctrl + R
    撤销 Ctrl + Z
    重做 Ctrl + Y

    基本语法

    文本样式

    语法 效果
    正常文字 正常文字
    *斜体文字* 斜体文字
    _斜体文字_ 斜体文字
    **粗体文字** 粗体文字
    ***斜体加粗*** 斜体加粗

    标题

    # 一级标题

    一级标题

    ## 二级标题

    二级标题

    ### 三级标题

    三级标题

    #### 四级标题

    四级标题

    ##### 五级标题
    五级标题
    ###### 六级标题
    六级标题

    链接

    示例1:

    
这是一个链接[h1985](https://h1985.vercel.app)

    显示结果如下：
    这是一个链接h1985

    示例2:

    <https://h1985.vercel.app>
    显示结果如下：
    https://h1985.vercel.app

    图片

    示例1:

    
![我的头像](https://qqhsx.github.io/images/avatar.png?v=1653910685113)

    显示结果如下：

    如果图片无法加载，默认显示”我的头像“文字

    示例2:

    
 ![我的头像](https://qqhsx.github.io/images/avatar.png?v=1653910685113 "h1985")

    显示结果如下：

    示例3：引用本地图片

    

    ![我的头像](/h1985.jpg "h1985")

    分割线

    在一行中使用 *** 或者 - - - 或者 ___ 都可以插入一个分割线，行内不允许有其他内容。

    
***


    
- - -


    
___

    代码块

    行内代码块（使用ESC键上的英文字符 ` 包裹代码），例如：

    
`printf()` 函数

    显示结果如下：

    printf() 函数
    代码块（缩进 4 个空格或是 1 个制表符）：

    使用空格定义代码块

    
    pubic static void main(String[] args) {
      System.out.println("Hello world");
    }

    其渲染结果如下：

    pubic static void main(String[] args) {
  System.out.println("Hello world");
}

    多行代码块（使用连续3个ESC键上的英文字符 ` 包裹代码）：

    
```
pubic static void main(String[] args) {
      System.out.println("Hello world");
    }
```

    其渲染结果如下：

    pubic static void main(String[] args) {
      System.out.println("Hello world");
    }

    引用

    单层引用：在被引用的文本前加上>符号，例如：

    
> 语出《论语》

    其渲染结果如下：

    语出《论语》

    嵌套引用：

    
> #引用1

    #引用1

    
>> ##引用2

    ##引用2

    列表

    无序列表（使用 *，+，- 表示）:

    
+ 1
- 2
* 3

    其渲染结果如下：

    • 1
    • 2
    • 3

    有序列表（使用数字紧跟一个英文句点表示）：

    
1. 1
2. 2
3. 3

    其渲染结果如下：

    1. 1
    2. 2
    3. 3
      无序列表和有序列表的列表内容前一定要有一个空格。

    表格

    
项目 | Value
-------- | -----
电脑 | $1600
手机 | $12
导管 | $1

    其渲染结果如下：

    项目 Value
    电脑 $1600
    手机 $12
    导管 $1 
    
| Column 1 | Column 2 |Column 2 |
|:--------:| -------------:|:-------------|
| centered 文本居中 | right-aligned 文本居右 | left-aligned 文本居左 |

    其渲染结果如下：

    Column 1 Column 2 Column 2
    centered 文本居中 right-aligned 文本居右 left-aligned 文本居左
