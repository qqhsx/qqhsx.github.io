    如何在 Windows 资源管理器预览窗格中预览 DWG 文件？

    问题：
    安装 ZWCAD 2017 后，您可能会发现 Windows 资源管理器的预览窗格中没有视图。
    原因：
    这是因为中望CAD 2017 没有提供部分.dll 预览功能。我们现在正在开发它，在它完成之前，您可以尝试以下解决方案。
    解决方案：

    1. 下载 FixPreivew
    2. 解压文件，然后以管理员身份运行FixPreview.exe；
    3. 如果失败，请下载Dependency Walker和DirectXRepair进行故障排除。
    4. 运行depends.exe，打开FxVistaPreview.dll查看相关的.dll模块。FxVistaPreview.dll需要和系统保持一致，也就是说，运行32位系统需要选择x86文件夹下的.dll，64位需要选择x64。如果有一些模块被标记为如下黄色，则需要转到步骤b。
    5. 如果在 Win7 中运行 DirectX Repair.exe，在 Win8 或 Win10 中运行 DirectX_Repair_win8_win10。然后点击右上角的按钮。该程序将自动运行以修复 .dll 模块。
    6. 完成后，您可以再次进入（1）。这一次，它应该工作。
      附加信息：
      您也可以手动编辑注册表以完成步骤（1）：
    7. 下载 FxVistaPreview.dll 并将其复制到路径中
      C:\Program Files\Common Files\ZWSoft Shared或 C:\Program Files\Common Files (x86) \ZWSoft Shared，以管理员身份运行 CMD，然后运行 ​​regsvr32 "C:\Program Files\Common Files\ZWSoft Shared\FxVistaPreview。 dll"或运行 regsvr32 "C:\Program Files (x86)\Common Files\ZWSoft Shared\FxVistaPreview.dll"
    8. 在运行中运行Regedit打开注册表编辑器，定位到Computer\HKEY_CLASSES_ROOT.dwg，如下图，可以看到ZWCAD.Drawing.2017（这一步不重要）
    9. 定位到计算机\HKEY_CLASSES_ROOT.dwg\shellex
      如果没有如上图所示的“shellex”，则创建一个名为“shellex”的文件。然后如果没有这个key，则创建一个名为{8895b1c6-b41f-4c1c-a562-0d564250836f}的新key，然后右键单击新key创建一个新的字符串值并将值数据设置为{6AD4617F-D034-4d66 -AE9E-143247611403}。

      就这样！
