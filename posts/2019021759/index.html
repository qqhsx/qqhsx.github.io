用Office Tool Plus下载和安装Office

发表于 2022-09-08 更新于 2022-09-09 分类于 办公软件安装 ， 办公软件安装教程 ， Microsoft Office 阅读次数： Valine： 本文字数： 1.1k 阅读时长 ≈ 1 分钟

引言

Office Tool Plus是什么？相信很多人对这个工具的认识都一无所知，今天我简答介绍一下，在介绍Office Tool Plus如何使用前，先带大家简答认识一下Office Tool Plus工具的主要用途！

Office Tool Plus简称OTP，是一款专业的Office官方镜像下载器，它不仅可以在线下载和安装Office产品的各个版本，除此之外还集成了各种实用性工具帮助用户部署和管理Office。还能对你系统已安装的Office软件进行激活。

有了Office Tool Plus工具后，你可以使用它下载微软官方的Office软件并进行安装和部署，你也可以将下载好的镜像制作成ISO文件分享给你的朋友和用户使用。无论你是个体还是团队，Offfice Tool Plus都是一款不错的首选工具。

下载教程

注意：Win7系统必须要SP1或更高版本才能使用，Office Tool Plus目前仅支持Windows平台，不支持其他操作系统（例如 Linux 或 macOS）

点击【立即下载】跳转至下载页面然后选择【包含框架 (ZIP 压缩包)】版本进行下载；

安装教程

安装前需要下载，用Office Tool Plus工具去下载Office镜像，具体操作如下：

压缩包文件解压，得到如下文件:

双击打开【Office Tool Plus】文件，然后同意许可协议即可；

点击【部署】选项然后选择产品，这里以Office2021版本为例；

选择产品，根据需要选择！

选择【应用程序】和【安装语言】，应用程序建议默认即可，安装语言选择【简体中文】

点击【安装设置】这里可设置镜像下载路径，不设置镜像会默认下载到程序包的Office Tool\Office\Data路径下；

点击【激活设置】将【自动接受许可条款】勾选；

部署设置这里有3个需要注意的地方，第1个是【体系结构】，有根据32位和64位系统定制的选项，根据电脑系统来选即可>第2个是【部署模式】，有下载、安装和创建ISO文件3个选项>第3个是【下载设置】，其他的都默认即可；

一切配置好后点击【开始部署】即可；

下载完后然后就可以开始安装和创建ISO镜像了；

将部署模式由之前的下载改为【安装】然后点击【开始部署】即可安装软件；

软件安装完毕后点击【关闭】即可；

制作ISO镜像教程

如果你是自己电脑使用没必要创建ISO镜像，但你要是想分享给你朋友和提取Office镜像，就可以将刚刚下载的文件通过【Office Tool Plus】制作成ISO镜像。

将部署模式由之前的安装改为【创建ISO文件】然后点击【开始部署】；

设置保存路径和命名，可以设置ISO文件保存路径和自定义命名；
